Obfuscating Autohotkey System Function Calls Like SubStr()

Obfuscation of built in autohotkey functions like SubStr() can only be done if you use Dynamic Obfuscation. This is necessary because this type of object can only be obfuscated with the use of %'s, it can not be done with straight obfuscation.

Use $DEFSYSFUNCS Command

Once you have set up basic Dynamic Obfuscation, there are just 2 commands that you will use to obfuscate built in functions.

;$OBFUSCATOR: $DEFSYSFUNCS: (list of functions, comma separated)
